how to install virastyar v4.0 beta for office 2016:

1- uninstall virastyar if installed
2- apply patch.reg file to registery
3- install virastyar v4 beta, download from https://sourceforge.net/projects/virastyar/files/Virastyar/4.0%20Beta/Virastyar%204%20Beta.zip/download
4- copy and replace these files into "C:\Program Files (x86)\Virastyar\Add-in"

	VirastyarWordAddin.dll
	VirastyarWordAddin.dll.backup
	VirastyarWordAddin.dll.manifest
	VirastyarWordAddin.vsto

5- run MS Word, if Virastyar add-in is not loaded. go to File > Options > Add-ins > Manage COM Add-ins, click Go > Enable Virastyar
6- done
